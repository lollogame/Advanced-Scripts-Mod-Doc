Basic Syntax

The scripting language uses Python-like syntax with indentation-based blocks and supports variables, loops, conditionals, and functions.

Comments

Use # for single-line comments:

# This is a comment
x = 10  # Inline comment

Indentation

Indentation (4 spaces or 1 tab) defines code blocks:

if x > 5:
    /say x is greater than 5
    x = x - 1

Variables & Types

Variable Assignment

name = "Player"
age = 25
height = 1.75
is_online = True
items = None

Supported Types

Type Example Description
String "hello" or 'hello' Text values
Integer 42 Whole numbers
Float 3.14 Decimal numbers
Boolean True, False Boolean values
List [1, 2, 3] Ordered collection
None None No value

String Interpolation

Use curly braces {variable} to interpolate variables in strings:

name = "Player"
/say Hello, {name}!

x = 10
message = f"The value is {x}"
/say {message}

Arithmetic Operations

a = 10
b = 3

sum_result = a + b      # 13
diff = a - b            # 7
product = a * b         # 30
quotient = a / b        # 3.33

Control Flow

If / Elif / Else

x = 5

if x > 10:
    /say x is greater than 10
elif x == 5:
    /say x is 5
else:
    /say x is less than 10

Comparison Operators

Operator Meaning Example
== Equal to x == 5
!= Not equal to x != 5
> Greater than x > 5
< Less than x < 5
>= Greater or equal x >= 5
<= Less or equal x <= 5

For Loop

for i in range(5):
    /say Number: {i}
    wait(10)

While Loop

counter = 0
while counter < 5:
    /say Counter: {counter}
    counter = counter + 1
    wait(10)

Break and Continue

for i in range(10):
    if i == 5:
        break
    if i == 2:
        continue
    /say {i}

Function Definition

def greet(name):

greet("Player")

Functions with Multiple Parameters

def give_items(player, item, amount):
    /give {player} {item} {amount}
    /say Gave {amount} {item} to {player}

give_items("@s", "diamond", 10)

Minecraft Commands

Executing Commands

Commands start with / and support variable interpolation:

player = "Dev"
/say Hello, {player}!
/tp {player} 0 64 0
/give {player} diamond 10
/summon minecraft:armor_stand ~ ~ ~

Tellraw (Colored Text)

/tellraw @a {"text":"Red Text","color":"red"}
/tellraw @a [{"text":"Multi","color":"gold"},{"text":"Color","color":"blue"}]

Built-in Functions

print(text)

Print debug message to console:

print("Debug message")
print(x)

wait(ticks)

Wait for specified ticks (1 tick = 50ms):

wait(10)   # Wait 500ms
wait(20)   # Wait 1 second

range()

Create number sequences:

# From 0 to n-1
for i in range(5):
    /say {i}  # 0, 1, 2, 3, 4

# From start to end
for i in range(1, 5):
    /say {i}  # 1, 2, 3, 4

# With step
for i in range(0, 10, 2):
    /say {i}  # 0, 2, 4, 6, 8

for i in range(10):
    /say [{i}] {a}
    temp = a
    a = b
    b = temp + b

Countdown Timer

def countdown(n):
    count = n
    while count > 0:
        /say Countdown: {count}
        count = count - 1
        wait(20)
    /say Blast off!

countdown(5)
